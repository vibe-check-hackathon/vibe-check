# seed + speed Ventures — Portfolio (enriched & verified)

Enriched and re-verified version of `seed-speed-portfolio.md`. Each company is
written as a compact **opportunity-card-style** record (mirrors
`sun/opportunity-card.md`) so it can be read the same way as a live deal card —
every entry carries an **Idea** (problem & product) and a **User** (who it is for),
plus geography, website, round history, and seed + speed's role.

## About the fund

seed + speed Ventures is the pre-seed/seed VC arm of the Maschmeyer Group, founded
by Dr. Carsten Maschmeyer. It backs mostly **B2B / enterprise software** across
Germany, Austria and Switzerland, with initial tickets of roughly €0.75M–€1.5M and
follow-on capacity up to ~€5M. (A separate US vehicle, MGV / Maschmeyer Group
Ventures, and the growth fund ALSTIN Capital are distinct entities and are *not*
covered here.)

## How to read each card

| Field | Meaning |
| --- | --- |
| **Idea** | The problem and the product — what it actually does |
| **User** | The target customer / who buys and uses it |
| **Sector** | Primary theme |
| **HQ** | Headquarters / primary geography |
| **Website** | Verified company domain |
| **LinkedIn** | Company page (best-effort slug — verify before outreach) |
| **Founded / founders** | Year and named founders where verified |
| **s+s role** | seed + speed's position (lead / participant / early backer) |
| **Rounds** | Known financing with amount, date, and lead / co-investors |
| **Status** | Active · Exit / acquired · Defunct |
| **Confidence** | ✅ verified this pass · ◻︎ from working list, not re-verified |

## Verification legend

- **✅ Verified this pass** — round size, date, lead/co-investors, HQ and founders
  cross-checked against at least one named public source (press release, tier-1
  startup press, or the fund's own portfolio page). seed + speed involvement
  explicitly confirmed unless noted.
- **◻︎ Not re-verified this pass** — carried over from the working list; treat
  sector/status as indicative and confirm before use.

- **User:** German retail banking consumers.
- **Sector:** FinTech (PFM / financial assistant) · **HQ:** Frankfurt/Eschborn (dwins GmbH)
- **Website:** finanzguru.de · **LinkedIn:** linkedin.com/company/finanzguru
- **Founded:** 2017 (dwins GmbH; publicly known from *Höhle der Löwen*).
- **s+s role:** Early/seed-stage backer.
