Step 1 / 3
scout-agent · running

🔍 Sourcing

The scout agent finds founders, verifies the company story, and progressively adds sourced facts to the opportunity card.

 sourcing.ts 
const card = new OpportunityCard("Nimbus AI");

for (const fact of await scout.discover()) {
  card.addFact({
    text: fact.summary,
    source: fact.origin,
  });
}

🤝 Negotiating

The negotiator interviews the founders, resolves the terms, and attaches the final term sheet for investment committee review.

 negotiate.ts 
const deal = await negotiator.interview(founders, {
  rounds: 3,
  draft: card.get("term_sheet_draft_v1.pdf"),
});

card.attach("term_sheet_FINAL.pdf", deal.terms);
card.setStatus("READY FOR IC ✅");
