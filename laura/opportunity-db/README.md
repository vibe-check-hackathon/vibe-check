# Opportunity DB — Maschmeyer Group Ventures (Real-World Demo Data)

Retrospective opportunity cards built from **public sources** about the
Maschmeyer Group's venture portfolio, compiled 2026-07-19. Purpose: show the
Opportunity OS card format working on real deals the challenge sponsor's group
actually made — each card is dated as if the 24-hour evaluation ran just before
the real, publicly announced investment.

## Integrity rules (read before editing)

1. **Facts are sourced.** Every material claim cites a public source in the
   card's evidence ledger, with honest trust scores; facts we could not verify
   are marked `claimed / unverified`, not guessed.
2. **No fabricated evaluations of real people.** Founder Scores, vibe-check
   signals, and assessment axes are `not assessed` — this system refuses to
   invent judgments about real, named founders. That is the product's own
   "unknown is not false" rule applied to ourselves.
3. **Interviews:** where a real public founder interview exists (YouTube),
   it is linked as what the Matching stage would study. Where none was found:
   **🔒 Confidential — no public interview found.**
